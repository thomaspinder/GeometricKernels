geometric_kernels#

The library root. The kernel classes are contained within the kernels package. They need to be paired with one of the space classes from the spaces package.

The frontends package contains kernel wrapper classes compatible with Gaussian process libraries like GPFlow, GPyTorch, and GPJax.

The lab_extras package contains our custom additions to LAB, the framework that allows our library to be backend-independent. This is an internal part of our library.
